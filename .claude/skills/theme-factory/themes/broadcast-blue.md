# Broadcast Blue

A professional, technical theme inspired by Cyanview's broadcast equipment brand identity. Deep navy foundations with vibrant cyan accents — engineered for clarity and trust.

## Color Palette

- **Signal Blue**: `#1e3a5f` - Primary dark, hero backgrounds, depth
- **Cyanview Blue**: `#006faf` - Brand primary, CTAs, active states
- **Bright Cyan**: `#6dc9f3` - Accents, highlights, gradient endpoints
- **Frost**: `#eef4f8` - Light background, card surfaces, breathing room
- **Slate**: `#334155` - Body text, readable contrast
- **Steel**: `#94a3b8` - Muted text, secondary information

## Typography

- **Headers**: Outfit (600-700 weight) — geometric, clean, technical
- **Body Text**: Outfit (400-500 weight) — consistent family, legible
- **Monospace**: Source Code Pro — code, badges, technical labels

## Dark Mode Variant

- **Deep Space**: `#0a0f1a` - Page background
- **Panel**: `#111827` - Card/surface background
- **Midnight**: `#040b18` - Footer, deepest layer

## Gradient Recipes

- **Hero**: `linear-gradient(160deg, #1e3a5f 0%, #162d4d 35%, #0f2440 65%, #0a1b35 100%)`
- **Footer**: `linear-gradient(180deg, #0e1f38 0%, #091428 40%, #061230 100%)`
- **Title accent**: `linear-gradient(135deg, #60a5fa 0%, #38bdf8 40%, #818cf8 100%)`

## Best Used For

Broadcast technology, engineering documentation, camera control systems, professional support portals, technical product sites.
