# NIO — Network I/O

The NIO is a network-based I/O device that gives you 16 GPIO channels for tally and control signals. Send tally over Ethernet, WiFi, or 4G — no dedicated cabling required.

![NIO — Network I/O](/img/productGfx/NIO/nio-wh.jpg)

## Key features

- **16 GPIO channels** — Flexible tally input and output
- **Multiple transport options** — Ethernet, WiFi, and 4G cellular
- **Stackable** — Use multiple NIO units when you need more than 16 GPIO
- **Network integration** — Connects to your RCP over standard IP networks

## When to use the NIO

Use the NIO when you need tally or GPIO control over a network connection, especially in remote production scenarios where running dedicated tally cables is impractical. The NIO pairs naturally with REMI setups using the [RIO +WAN](/docs/products/rio).
