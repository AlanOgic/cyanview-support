# Tally box PCB replacement

## Abstract

This document explains step by step how to replace the PCB in a tally box.

This replacement is straightforward and only requires a Phillips screwdriver (PH0).

## Replacement procedure

1. Place the tally box flat on a surface and remove the four screws using a Phillips screwdriver.

![Open the tally box](/img/restricted/troubleshooting/tally/tallybox_pcb2.jpg)

2. Open the tally box.

![Tally box opened](/img/restricted/troubleshooting/tally/tallybox_pcb3.jpg)

3. Remove the PCB from the enclosure by pulling it upward, holding it on the side opposite the connectors.

![Remove the PCB](/img/restricted/troubleshooting/tally/tallybox_pcb4.jpg)

4. Install the new PCB by reversing the previous step. Insert it at an angle so that both connectors fit correctly into their openings. Make sure the alignment pin on the side opposite the connectors goes properly through the PCB hole.

![Install the new PCB](/img/restricted/troubleshooting/tally/tallybox_pcb5.jpg)

5. Set the selectors so that they are both oriented inward.

![Set the selectors](/img/restricted/troubleshooting/tally/tallybox_pcb6.jpg)

6. Set the two mechanical sliders in the same way, so that they are also oriented inward.

![Set the mechanical sliders](/img/restricted/troubleshooting/tally/tallybox_pcb7.jpg)

7. Hold the enclosure vertically with the selectors facing downward, and fit the two halves together. Check the alignment between the selectors and the mechanical sliders to ensure everything fits correctly. This is the most delicate step.

![Close the enclosure](/img/restricted/troubleshooting/tally/tallybox_pcb8.jpg)

8. Verify that the enclosure is correctly assembled, then reinstall the four screws.

![Reinstall the screws](/img/restricted/troubleshooting/tally/tallybox_pcb9.jpg)
