## Home screen

![Cyanview RCP home screen](/img/Manual/RCP/RCP-home-screen.BMP) ![Cyanview RCP home screen alternate view](/img/Manual/RCP/RCP-home-screen1.BMP)

## Menu organization

Press the **MENU** button (top right) to enter the menu system. Settings are organized into categories and sub-menus.

Once inside a sub-menu or leaf:

- **Short press MENU** — hide or show the menu overlay
- **Long press MENU** — return to the main menu
- **Press BACK** (right white button) — go up to the parent menu
- **Press ⏴ / ⏵** (3rd and 4th white buttons from the left) — navigate between leaves of the same sub-menu
