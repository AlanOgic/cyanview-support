Control the Panasonic AG-CX350 camcorder over IP — either via wired Ethernet or the camera's built-in Wi-Fi.

:::info
Keep the camera firmware up to date. The latest firmware for the AG-CX350 is version [7.20-00](https://eww.pass.panasonic.co.jp/pro-av/support/content/download/EN/ep2main/soft/upgcx350_e.htm).
:::

## Supported control

| Control | Notes |
| :------ | :----- |
| Iris | Manual and AUTO iris (speed, sensitivity) |
| Gain | In dB, manual and AUTO (max gain and EV settable) |
| Shutter | Speed or ECS |
| ND [^1] | ND can be changed but cannot be recalled from scene files — protocol allows control by increments only |
| White balance | R/B gain, master gain, color temp |
| Black balance | R/B gain, master black |
| Saturation and Hue | |
| Knee | Point and slope, auto |
| Black gamma | Master and range |
| Detail and coring | Detail level/freq/mix, coring level, skin detail (skin 1, 2, 3) |
| OETF | HD / SD / FILMLIKE1 / FILMLIKE2 / FILMLIKE3 / FILM-REC / VIDEO-REC / HLG |
| Color space | NORMAL1 / NORMAL2 / FLUO / CINELIKE |
| Matrix and multimatrix | Multimatrix with 16-vector hue and saturation controls |
| Zoom and Focus | Speed control; position is read-only |
| Other | Pattern, ATW, AWB, ABB |
| Menu navigation | ON/OFF, enter/back, up/down, display toggle |

![CX350 RCP UI overview](/img/Integrations/Panasonic/CX350/RCP-UI/overview.gif)

[^1]: ND can be changed but cannot be recalled from scene files as the protocol only allows control by increments.

## Connection

Connect the camera to your network using wired Ethernet or built-in Wi-Fi.

![CX350 wired connection](/img/Integrations/Panasonic/CX350/Panasonic-CX350-connect.png)

![CX350 Wi-Fi connection](/img/Integrations/Panasonic/CX350/Panasonic-CX350-connect-wifi.png)

## Configuration

### Camera

Set your network to static or DHCP in `NETWORK > LAN PROPERTY > IPV4 SETTING`:

![CX350 IP setup](/img/Integrations/Panasonic/CX350/CX350-setup-IP.png)

Enable remote control in `NETWORK > IP REMOTE` (set to `ENABLE`):

![CX350 remote control setup](/img/Integrations/Panasonic/CX350/CX350-setup-remote.png)

Create an account (login/password) in `NETWORK > IP REMOTE`. If no account exists in `ACCOUNT LIST`, add one in `USER ACCOUNT`:

![CX350 account setup](/img/Integrations/Panasonic/CX350/CX350-setup-account.png)

### RCP

1. Select brand `Panasonic` and model `AG-CX350`.
2. Enter the camera `IP` address.
3. Enter the `login` and `password` from the account you created in `USER ACCOUNT`.

![CX350 RCP camera setup](/img/Integrations/Panasonic/CX350/RCP-cam-setup.png)
