# Sony PTZ

This guide describe how to configure Sony PTZ cameras

<img src="/img/Integrations/Sony/brc-x1000.png" width="300"/>

## Compatible PTZ

* BRC-X1000
* BRC-X400
* BRC-H900
* BRC-H800

## Controls

You can control:
* Exposure
* White balance
* Black balance
* Pan/Tilt/Zoom/Focus
